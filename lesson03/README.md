# Lesson III: Face recognition

## Realtime face detection

Create virtual environment
```bash
$ virtualenv -p python3 facerecognition
$ source facerecognition/bin/activate
```

Install Numpy
```bash
$ pip install numpy
```

Install OpenCV library
```bash
$ pip install opencv-python
```
